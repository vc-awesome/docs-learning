## 设置

底部 ActionButton ：

> 点击底部操作按钮“ⵈ”，打开命令模式。

- Toggle block comment - *切换块注释*
- Toggle comment - *切换注释*
- Remove line - *删除行*
- Find - *查找* <kbd>Ctrl</kbd> + <kbd>F</kbd>
- Select line to start - *选择一行到行首*
- Select line to end - *选择一行到行尾*
- Fold all - *全部折叠*
- Go to line... - *跳转行...* <kbd>Ctrl</kbd> + <kbd>G</kbd>

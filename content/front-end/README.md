# 前端

## 简介

> 网页编程采用的是 HTML + CSS + JS 组合。
>
> - HTML 是用来描述当前这个页面的结构。
>
> - CSS 用来描述页面的样式。
>
> - JS 通常是用来处理这个页面和用户的交互。

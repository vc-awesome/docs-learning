## DNS

?> DNS（Domain Name System）域名系统，简单点来说，就是当我们在浏览器输入一个网址，需要有一台DNS服务器帮我们解析成IP地址，我们才能打开这个网站。那么，DNS服务器解析的速度快慢，直接影响到我们访问网站的快慢。
